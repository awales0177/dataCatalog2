# Data directory

Static JSON and small modules used by the catalog UI.

## Files

- `models.json` — Catalog data model definitions
- `toolkit.json` — Toolkit metadata
- `dataAgreements.json` — Agreement-related data
- `reference.json` — Reference data
- `theme.json` — Theme tokens
- `toolkitEvalIcons.js` — Icons for toolkit evaluation states

Other notebooks or assets in this folder are for local authoring unless imported by the app.
